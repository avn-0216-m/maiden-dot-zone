Welcome to the Maiden Zone

Domain

maiden.zone was purchased from Namecheap, which true to its name offered a price $10 to $20 dollars cheaper than other registrars. It also includes free WHOIS privacy so I don't have to dox myself just to make a fun website.

Hosting

My first choice for hosting was recommended by a friend who also uses it, DigitalOcean, but site admins locked my account immediately after I created it for reasons unknown, so I went with Linode instead. I am currently using a Linode Shared CPU (2 GB) to host this website.

Software

I use Nginx to serve web pages. It's easy to set up and doesn't require much configuration.
